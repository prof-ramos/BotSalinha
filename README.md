# BotSalinha

Assistente de Direito e Concursos Públicos para Discord.

## Funcionalidades

- Respostas sobre direito brasileiro
- Auxílio em estudos para concursos
- Histórico de conversas
- RAG com documentos jurídicos (em desenvolvimento)

## Tecnologias

- Python 3.12
- Discord.py
- Convex (backend cloud)
- OpenAI GPT-4o-mini

## Uso

```
!ask <pergunta> - Faça uma pergunta
!ping - Verifique a latência
!ajuda - Mostra ajuda
```

## Desenvolvimento

Desenvolvido com ❤️ para a comunidade de concurseiros.
